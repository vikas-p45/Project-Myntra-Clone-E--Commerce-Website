# 🛍️ Myntra Clone - E-commerce Website

A full-stack Myntra-inspired e-commerce web application built using modern technologies like React and Redux. This project demonstrates dynamic UI, state management, and backend API integration.

---

## 🚀 Features

- 🛒 Browse products with dynamic data
- 🔍 View product details
- ➕ Add to cart functionality
- 🔄 State management using Redux
- 📱 Fully responsive design
- ⚡ Fast and optimized performance

---

## 🛠️ Tech Stack

**Frontend:**
- React.js
- Redux
- HTML
- CSS
- JavaScript

  Backend:
  Node.js
  Express.js

  Data Handling:
  JSON (items.json)

 📂 Project Structure

Myntra-Clone/
│── frontend/        # React + Redux code
│── backend/         # Node.js APIs
│── data/            # JSON data
│── components/      # React components
│── store/           # Redux store
│── app.js           # Backend entry point

 ⚙️ How to Run Locally

git clone https://github.com/vikas-p45/Project-Myntra-Clone-E-Commerce-Website.git

cd Project-Myntra-Clone-E-Commerce-Website

npm install

npm start
